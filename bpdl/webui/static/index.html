Download history

"Verify library" checks whether each downloaded track's file still exists where it was saved. If you move files elsewhere afterwards, missing entries here are expected and fine — that's what keeps already-downloaded tracks from being re-downloaded. Only use "Remove missing entries" if you keep files in place and want to clean up genuinely lost ones.
